# range()
```py
for i in range(1,11):

    print(i)
```

this will print from 1 to *10*

# using steps in range()
```py
for i in range(1,11,3):

    print(i)
```

> 1
> 4
> 7
> 10

# sum of evens only in 1 to 100
```py
sum = 0

for number in range(2,101,2):

    sum = sum + number

print(sum)
```
